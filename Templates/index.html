Funções do Site

  • Recebimento de Webhooks

    A função principal do site é receber os webhooks do sistema de pagamento e tomar decisões com base no status do pagamento.

  • Liberar Acesso ao Curso

    Quando um pagamento é aprovado, o sistema deve liberar o acesso do cliente ao curso.

  • Enviar Mensagem de Boas Vindas

    Após a aprovação do pagamento, o sistema deve enviar uma mensagem de boas vindas ao cliente.

  • Enviar Mensagem de Pagamento Recusado

    Se um pagamento for recusado, o sistema deve enviar uma mensagem ao cliente informando sobre a recusa.

  • Remover Acesso ao Curso

    Se um pagamento for reembolsado, o sistema deve remover o acesso do cliente ao curso.

  • Registro de Webhooks e Tratativas

    Todos os webhooks recebidos e as ações realizadas pelo sistema devem ser registrados para futuras consultas.

  • Autenticação de Login

    Acesso ao site é permitido apenas para usuários autenticados. A criação de conta requer o token de autorização.

  • Tela de Tratativas por Usuário

    Os usuários têm acesso a uma tela onde podem ver todas as tratativas realizadas pelo sistema para cada usuário.

  • Pesquisar Tratativas

    Os usuários podem pesquisar tratativas por usuário e visualizar o histórico das ações realizadas pelo sistema para cada usuário.
